Secondary Color Palettes

The Secondary Palette should be used as a guide for the tone of a color. We intend for the secondary palette to be ever evolving, offering an endless palette to choose from. The main use of the secondary palette is in the creation of icons and illustrations that will be used across the board to help demonstrate the core benefits of Frontify products. We will also be using the secondary palette to create unique secondary brands for each one of Frontifys' products. Percentage tints can be used of any of these colours.

Colour Usage

We've chosen the Frontify Yellow to be our main product color for Frontify Style guide. Use the Frontify Blue as a neutral color for contrasting reasons.

Frontify has 5 main colors, try to balance the usage evenly and consider the product usage:

  • Teal deep: headings, buttons, borders, icon background, background to section of greater focus
  • Yellow: accents, primary action buttons, warnings, search result borders, tile borders, form selected form elements, aread of greater user focus
  • Red: Error section background, input error messages and border
  • Green: Success area background
