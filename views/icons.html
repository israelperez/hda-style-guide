Icons

All icons on the site are part of a custom font library that is managed at IcoMoon using their online app. Our library is a blend of FontAwesome, IcoMoon (free) and custom icons. IcoMoon uses the browsers local cache to store icons used. As such you will need to upload the project file (selection.json) provided in the download below to the website upon first use. Included in the download is a copy of the font files in multiple formats. Note that the demo.html doesnt work due to naming method used which contains a SCSS variable.

developer notes

Accessible Icons

Icons used for pure decoration or visual styling

If you're using an icon to add some extra decoration or branding, it does not need to be announced to users as they are navigating your site or app aurally. Additionally, if you're using an icon to visually re-emphasize or add styling to content already present in your HTML, it does not need to be repeated to an assistive technology-using user. You can make sure this is not read by adding the aria-hidden="true" to your Font Awesome markup.

examples

<i class="hda-icon_logo-small" aria-hidden="true"></i>
an icon being used as pure decoration

Icons that are for interaction

If you're using an icon to convey meaning (rather than only as a decorative element), ensure that this meaning is also conveyed to assistive technologies. This goes for content you're abbreviating via icons as well as interactive controls (buttons, form elements, toggles, etc.).

In the case of focusable interactive elements, there are various options to include an alternative text or label to the element, without the need for any visually hidden <span> or similar. For instance, simply adding the aria-label attribute with a text description to the interactive element itself will be sufficient to provide an accessible alternative name for the element. If you need to provide a visual tooltip on mouseover/focus, we recommend additionally using the title attribute or a custom tooltip solution.

<a href="url" aria-lable="Share via facebook">
    <i class="hda-icon_facebook" aria-hidden="true"></i>
</a>
A clickable icon for sharing content via facebook.

Note: there is no title attribute on the link. Do not use title attributes on anchor tags. Find out more here

Suppot classes

Fixed width Icons

Use hda-icon__fixed-width to set icons at a fixed width. Great to use when different icon widths throw off alignment. Especially useful in things like nav lists & list groups.

Stacked Icons

To stack multiple icons, use the hda-icon__stack class on the parent, the hda-icon__stack-1x for the regularly sized icon, and hda-icon__stack-2x for the larger icon. fa-inverse can be used as an alternative icon color. You can even add larger icon classes on the parent to get further control of sizing.

right caret icon stacked on top of the circle-o-notch

<div class="hda-icon__stack">
    <i class="hda-icon_circle-o-notch hda-icon__stack-2x"></i>
    <i class="hda-icon_caret-right hda-icon__stack-1x"></i>
</div>

Note: As per HDA's CSS standards, do not use multiple classes on an element. Find out more here

Animated Icons

Use the hda-icon__is-spin class to get any icon to rotate, and use hda-icon__is-pulse to have it rotate with 8 steps.

<i class="hda-icon_circle-o-notch hda-icon__is-spin"></i>
<i class="hda-icon_caret-right hda-icon__is-pulse"></i>

Note: As per HDA's CSS standards, do not use multiple classes on an element. Find out more here
