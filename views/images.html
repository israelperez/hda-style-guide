home page

The home page has 3 responsive sizes and at each size consideration must be made as to what will be visible as the image grows and more get cropped as the browser window gets larger

Mobile

At the mobile size the image is entirely visible and has a width to match the next break point. ie tablet at 760px. It will resize to fit the users browser width.

Dimensions
Width: 760px
Height: match aspect ratio

Visible
Width: 100%
Height: 100%

Tablet

The tablet version has a width of the desktop breakpoint ie. 1030px. It will resize to match the users browser size. However not all the image is visible. the viewable area is in the center of the photo with a height of 400 pixels from the center no mater the size of the browser. The width of the viewable area is 100%, however it will be partially obscured by Hero text area that was a width of 300px and has a transparency of 90%.

Dimensions
Width: 1030px
Height: match aspect ratio

Visible
Width: 100% - 300px
Height: 400px

Text area
Opacity: 90%
Width: 300px
Height: 400px

Desktop

The desktop version has a width of 1920 pixels and it will resize so that 100% of the width is visible. However only 400 pixel of the height is ever visible no mater the size of the browser. This means that the wider the users browser is the more narrow the visible photo is. Alos there is hero text in a box that sits the right hand side of the layout with a width of 400 pixles and a height to match the content that completely blocks the image.

Dimensions
Width: 1920px
Height: match aspect ratio

Visible
Width: 100%
Height: 400px

Text area
Width: 400px
Height: match text

Article and main content pages

Hero image

The most used image type is this one. there are 3 uses and 2 versions for the image. All versions maintain the aspect ratio of the desktop version.

This type of image is used on the following pages

  • Article pages
  • Theme pages
  • Medicines landing page
  • A-Z pages
  • Corporate pages
  • Generic pages

Mobile version

The mobile version of the hero image is used as the heading for the tiles that appear on themepages. This is regardless of the device or browser width as theme tiles will never be wider than 760px.

This version is also used as the hero image for the above mentioned pages and is only visible within the mobile breakpoint

Dimensions
Width: 760px
Height: 217px

Tablet / Desktop version

Used as the hero image for the above mentioned pages, this version is visible on the tablet and desktop breakpoints

Dimensions
Width: 1050px
Height: 300px
