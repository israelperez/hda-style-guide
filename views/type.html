Our Font Families

Our font families should be used for all communications. This to ensure the consistent look and feel of all literature off and online.

Open Sans should be used for all header, sub header text, and footer all other copy (body, quotes etc). To ensure a consistent look, make sure fonts are used in a consistent manner and the weights of the font have been considered.

developer notes

The dorado framework uses a base font size of 16px applied to the html and body elements from which the heading font-sizes are derived using a multiplcation factor of 1.2 for mobile and tablet and 1.333 for desktop. line-heights are derived from the resulting font-size by a factor of 1.2. This ensures that If the base font size changes, all fonts will adapt. it also ensures that our fonts scale uniformly.

for example (mobile and tablet):

starting with H5, which is the same size as normal font (ie 16px)
H4 font-size = <base font size> x <mulitplier>
H4 font-size = $font-size-base x $heading-device-factor
H4 font-size = 16px x 1.2 (for mobile/devices)
H4 font-size = 19.2px (or 1.2rem relative to body)

for a H3
H3 font-size = 16px x 1.2 x 1.2
H3 font-size = 23.04px (or 1.44rem relative to body)

for a H2
H2 font-size = 16px x 1.2 x 1.2 x 1.2
H2 font-size = 27.648px (or 1.728rem relative to body)

for a H1
H1 font-size = 16px x 1.2 x 1.2 x 1.2 x 1.2
H1 font-size = 33.1776px (or 2.0736rem relative to body)

H6 is done differently as it needs to be smaller than H5 and Body font-sizes
H6 font-size = 16px / 1.2
H6 font-size = 13.3333px (or 0.83333rem relative to body)

Desktop font-sizes are calcuated in the same way but using a scalling factor or 1.333 $heading-desktop-factor

text related sass variables

_colours.scss

  • $font-heading-colour = $brand-primary = $hda-teal-deep = #1e4759
  • $body-text-colour = $hda-grey-darkest = #333333
  • $text-muted = $hda-grey-dark = #999999
  • $text-colour-on-dark = #ffffff
  • $text-colour-on-light = $hda-grey-darkest = #333333

_typography.scss

  • $font-family-sans-serif = "Open Sans", "Helvetica Neue", Helvetica, Arial, sans-serif
  • $font-family-base = $font-family-sans-serif !default;
  • $font-size-root = 16px
  • $font-size-root-print = 12px
  • $font-size-base = 1rem
  • $line-height = 1.5
  • $line-height-headings = 1.2
  • $line-height-print = 1.1
  • $font-weight-root = 400
  • $font-weight-strong = 600
  • $font-weight-heading = 400
  • $font-weight-heading-small = 600
  • $dt-font-weight = 300
